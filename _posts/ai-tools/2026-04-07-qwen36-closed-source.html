知乎上有个问题：Qwen 3.6 转向闭源，怎么看？

底下吵翻了天。有人骂阿里背叛开源精神，有人分析这是受 DeepSeek 所迫，还有人搬出"开源挣名声闭源挣 money"的老话。

吵得都对。但我想先说一件很多人忽略的事：

Qwen3.6 Plus 在 OpenRouter 上是免费用的。

你不用下载，不用部署，点开就能跑。百万级上下文，65.5KB 最大输出，63 token 每秒。

所以问题根本不是"闭源 vs 开源"这么简单。

DeepSeek V4 鸽了，行业等来了什么

知乎 @jacky（265赞）说得最直接：

DeepSeek V4 整个三月都没发。

这事本来一年前就该搞的——DeepSeek 一开源，大家发现闭源打不过，那就开源先赚吆喝。然后就等着：DeepSeek 发新模型，继续吊打各厂商，那就继续开源；如果不发或者发了个一般的，那就赶紧圈钱。

结果 V4 真的鸽了。

行业等了一个季度，等来的是 Qwen 3.6 Plus 直接闭源。

@王欣宇（340赞）更是一针见血：Qwen 这次是受 Minimax 和 Mimo 的启发——先以预览版的名义推出一个独占的免费闭源版本，再去跟 Kilo code 这类合作，用来刷 OpenRouter 的榜单，等目的达到了，最终还是要开源的。

翻译一下：先闭源跑个分，占个心智，然后再开源卖生态。

这套路眼熟吗？

@GYK（50赞）翻出了 Qwen 的历史：DashScope API 产品线（qwen-turbo/plus/max）一直都是闭源的。Qwen Max 始终领先开源版本一到四个季度。这是老传统了，不是突然背叛。

这不是背叛，这是策略

知乎 @平凡（202赞）给了一个很冷静的框架：

最大型号闭源（暂时），小型号变种开源，是一种科学的营销策略。

Reddit 上有老哥说得很公道：应该鼓励中国前沿实验室把最大最复杂的模型闭源，只要满足两个条件就够了——

  • (a) 开源前沿模型的小版本；
  • (b) 在前沿模型被弃用后，开源旧版本。

因为现实是：前沿模型研发需要商业化支撑。如果要求实验室在和 OpenAI/Anthropic 竞争的同时，还永远免费提供一切技术，这本身就不现实。

@阿里云 EMNLP25 的官方说法也是这个调子：旗舰大模型闭源，只开放小参数模型供社区学习参考。

这不是阿里一家的事。

@长风破浪（95赞）说的大趋势是真的：从 GLM-5.1，到 MiniMax-M2.7，再到 Qwen 3.6，全都在缓开慢开。

@Gahole（74赞）补充了另一家：智谱可以闭源了，GLM-5 打出名气，GLM-5.1 闭源赚钱，最新版闭源，等有更强模型了，再把旧版开源。

开源是为了挣名声，闭源是为了挣钱。除非你有 Claude 那么牛逼的模型，不然还是先开源刷榜，等大家都知道你了，再卖 Code Plan。

三类分层，Qwen 到底关了什么

@智械降神（170赞）把 Qwen 3.6 的分层说得很清楚：

第一类：MAX、Plus（397B-A17B）不开源

第二类：122B-A10B、27B 只开源 chat 版本

第三类：35B-A3B、9B、4B、2B、0.8B 开源 chat 和 Base 两个版本

不止 3.6，Qwen 3.5-Omini 的三个尺寸也都没开源。@智械降神 甚至说，他不知道 9B 的 Omini-Light 闭源的意义在哪里——但这件事本身，其实说明的恰恰不是阿里"背叛开源"，而是闭源和开源之间的边界，在 Agent 时代正在被重新划定。

@清顺（169赞）提了一个更值得关注的点：

现在的重心不该是 Qwen 3.5-Omni 吗？它可能会对很多 AI 基础设施产生冲击。

我们真正该担心的事

@李永峰（59赞）给了一个很反直觉的角度：

如果中国一线大模型真的走向闭源，这对英伟达是个极大利空。

逻辑链是这样的——

国外大模型推理侧主要部署在亚马逊云、微软云、谷歌云三大云厂商上，他们都有自有的推理芯片。未来采购英伟达算力会大幅减少。英伟达讲的故事是"主权算力"：给中东、欧洲等地区卖显卡，服务当地政府。如果中国一线大模型不开源了，他们买显卡干嘛？不如直接买 Token 呢。

他推测英伟达会自己去做开源大模型，而且开源协议会约定这类模型只能运行在英伟达的卡上。

这个逻辑有没有道理另说，但它提醒了一件事：开源闭源的博弈，从来不只是技术问题，而是整个 AI 产业链利益格局的重构。

所以，开源社区输了？

我不这么看。

@平凡 说了另一件事：大家可以去 Qwen 的 Hugging Face 上看，他们发布下一代模型之后，基本上都会开源小版本。版本更迭非常快，几周甚至几天就有一个新版本出来。3.6 Plus 虽然现在还没开源，但"不久的以后"大概率会开源。

而且说到底，开发者真正需要担心的是模型本身开不开源吗？

腾讯新闻的一篇分析说了一个更本质的判断：

当 token 成本足够低，几乎所有应用创意都变得可行。真正的转折点不是 token 便宜到什么程度，而是你开始忘记模型的存在——它只是你工作流中自然的一部分。

真正重要的不是哪个公司开源了哪个模型，而是整个生态是否保持活力：小模型能不能持续变强？工具链是不是越来越好？本地部署的成本是不是越来越低？

Qwen 3.6 闭源了。

但 0.6B 到 72B 的各种尺寸，依然在开源。Qwen 家族依然是你能本地部署的最强中文基座之一。

开源精神的核心，从来不是"每一家公司都必须开源每一个模型"。

而是：不管谁用什么策略，最终整个生态保持开放、竞争、创新。

从这个标准看，中国的 AI 开源生态，比两年前强太多了。
