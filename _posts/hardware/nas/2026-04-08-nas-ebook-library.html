NAS电子书管理：搭建个人数字图书馆

2026-04-08 · 数码硬件/NAS存储 · 🔖 NAS, 电子书, Calibre, 书库, 阅读, 图书管理

电子书越来越多，找不到想看的书？不同设备间图书同步麻烦？

用NAS搭建个人数字图书馆，统一管理所有电子书，随时随地在线阅读！

一、方案介绍

Calibre-Web

Calibre-Web是基于Calibre数据库的Web界面，功能完善且美观：

  • 支持海量图书管理
  • 在线阅读（EPUB、PDF、TXT）
  • 格式转换（EPUB↔MOBI↔AZW3）
  • 推送到Kindle
  • 用户权限管理
  • OPDS订阅（阅读器同步）
  • 多语言支持（含中文）

其他方案对比

方案 特点 适合场景
Calibre-Web 功能全面、界面美观 个人书库、多用户
Calibre桌面版 功能最强、需PC 图书整理、格式转换
Kavita 漫画/杂志优化 漫画收藏
Komga 专注漫画 漫画管理
简单文件夹 零配置 少量图书

二、Docker安装Calibre-Web

准备工作

创建目录

mkdir -p /volume1/docker/calibre/books
mkdir -p /volume1/docker/calibre/config

获取UID/GID（避免权限问题）

# SSH连接到NAS
ssh admin@NAS_IP

# 查看当前用户UID/GID
id

# 记录输出，如：uid=1024(admin) gid=100(users)

运行容器

docker run -d \
  --name=calibre-web \
  --restart=always \
  -p 8083:8083 \
  -v /volume1/docker/calibre/books:/books \
  -v /volume1/docker/calibre/config:/config \
  -e PUID=1024 \
  -e PGID=100 \
  -e TZ=Asia/Shanghai \
  technosoft2000/calibre-web:latest

参数说明：

  • -p 8083:8083：Web访问端口
  • -v /books：图书库存储路径
  • -v /config：配置文件路径
  • -e PUID/PGID：用户权限ID
💡 镜像选择：technosoft2000/calibre-web功能最全，支持在线阅读和格式转换。linuxserver/calibre-web更轻量但没有转换功能。

初始化配置

步骤1：访问安装页面

  1. 浏览器访问 http://NAS_IP:8083
  2. 进入Calibre-Web配置向导

步骤2：配置数据库路径

  • 数据库路径：/books（固定写法）
  • 如果没有数据库，会自动创建

步骤3：设置管理员账号

  1. 默认管理员：admin
  2. 默认密码：admin123
  3. 登录后立即修改密码

三、基本配置

切换中文界面

  1. 登录后点击右上角【admin】
  2. Language选择【简体中文】
  3. 点击【保存】

开启上传功能

  1. 进入【管理权限】→【编辑基本配置】
  2. 勾选【启动上传】
  3. 保存设置

配置格式转换

  1. 进入【基本配置】→【扩展程序配置】
  2. Calibre电子书转换器路径：/usr/bin/ebook-convert
  3. Kepubify电子书转换器路径：/usr/bin/kepubify
  4. 保存设置

开启用户注册

  1. 【管理权限】→【编辑基本配置】
  2. 勾选【启动注册】
  3. 选择【使用邮箱或用户名】
  4. 配置SMTP邮箱（用于发送验证邮件）

四、图书管理

添加图书

方式1：Web上传

  1. 点击【上传】按钮
  2. 选择电子书文件（EPUB、PDF、MOBI等）
  3. 自动提取元数据

方式2：批量导入

  1. 将电子书放入/books文件夹
  2. 进入【管理权限】→【图书列表】
  3. 点击【扫描书籍】

方式3：Calibre桌面版同步

  1. 电脑安装Calibre桌面版
  2. 书库位置设置为NAS的/books文件夹
  3. 在电脑上整理图书，自动同步到NAS

编辑元数据

点击图书→【编辑元数据】：

  • 书名、作者、出版社
  • ISBN、出版日期
  • 书籍简介
  • 封面图片
  • 标签分类

创建书架

  1. 点击【书架】→【新建书架】
  2. 输入书架名称（如"技术书籍"、"小说"）
  3. 将图书添加到书架

五、在线阅读

浏览器阅读

  1. 点击图书封面
  2. 选择【在线阅读】
  3. 支持EPUB、PDF、TXT格式

阅读设置

  • 字体大小调整
  • 背景颜色（白天/夜间模式）
  • 行间距、页边距
  • 书签功能
  • 阅读进度同步

格式转换

转换为Kindle格式：

  1. 点击图书→【转换书籍】
  2. 选择目标格式（MOBI/AZW3）
  3. 点击【开始转换】
  4. 转换完成后下载

六、推送到Kindle

配置Kindle邮箱

  1. 登录亚马逊账户
  2. 进入【管理我的内容和设备】
  3. 找到Kindle设备的邮箱地址（如xxx@kindle.cn）

配置Calibre-Web

  1. 进入【管理权限】→【编辑基本配置】
  2. 配置SMTP邮箱（用于发送）
  3. 设置已认可的发件人邮箱

推送图书

  1. 点击图书→【推送到Kindle】
  2. 选择目标Kindle设备
  3. 自动转换为MOBI格式并发送
  4. Kindle联网后自动下载

七、OPDS订阅

什么是OPDS

OPDS（Open Publication Distribution System）是一种电子书分发协议，支持在阅读器中直接浏览和下载图书。

获取OPDS地址

  • OPDS地址：http://NAS_IP:8083/opds
  • 需要登录验证

阅读器配置

KyBook（iOS）

  1. 打开KyBook
  2. 添加书库→选择OPDS
  3. 输入Calibre-Web的OPDS地址
  4. 输入用户名密码

FBReader（Android）

  1. 打开FBReader
  2. 书库→添加网络书库
  3. 选择OPDS，输入地址

其他支持OPDS的阅读器

  • Marvin（iOS）
  • MapleRead（iOS）
  • Aldiko（Android）

八、用户权限管理

创建用户

  1. 【管理用户】→【新建用户】
  2. 设置用户名、密码、邮箱
  3. 分配权限（查看/上传/编辑/删除）

权限设置

权限 说明
查看书籍 浏览和在线阅读
下载书籍 下载电子书文件
上传书籍 添加新图书
编辑书籍 修改元数据
删除书籍 删除图书
匿名浏览 无需登录查看

访客模式

  1. 【管理权限】→【编辑基本配置】
  2. 勾选【允许匿名浏览】
  3. 设置访客可见的书架

九、高级功能

豆瓣插件（获取元数据）

  1. 进入【管理权限】→【插件】
  2. 安装豆瓣元数据插件
  3. 编辑图书时点击【从网络获取元数据】

自动标签

十一、常见问题

Q：Calibre-Web和Calibre桌面版什么关系？

A：Calibre-Web使用Calibre的数据库格式，可以共用同一个书库。桌面版适合整理图书，Web版适合在线访问。

Q：没有数据库怎么办？

A：首次运行会自动创建空数据库。也可以先在电脑上安装Calibre桌面版创建书库，再复制到NAS。

Q：权限错误怎么解决？

A：检查PUID/PGID设置是否正确；确保books和config文件夹有读写权限。

Q：在线阅读支持哪些格式？

A：支持EPUB、PDF、TXT。MOBI/AZW3需要转换为EPUB才能在线阅读。

Q：如何批量导入图书？

A：将图书文件复制到books文件夹，然后在管理界面点击"扫描书籍"。

Q：书库可以多大？

A：取决于NAS硬盘容量。Calibre支持管理数万本图书。
